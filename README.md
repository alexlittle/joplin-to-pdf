# joplin-to-pdf


For converting a set of markdown files into a pdf. Written specifically for my revision notes that I have in Joplin, to 
convert to an indexed pdf that I can take into an open book exam. Although written for Joplin, it should also work for 
any directory of markdown files. 


Requires:
- pandoc - `sudo apt install pandoc`


Steps:

1. (optional) Export notebook from Joplin (using Export $\to$ markdown)
2. Run generate.py


Gotchas:

- Very picky about Latex/Katex format
